Mudae Harem Organizer Final - Local Icons

Organized folders:
- app/       Browser favicon / app icon. Replace app/favicon.png with your own PNG icon.
- keys/      Local key icons used by key stats.
- kakera/    Local kakera icons used by cards/edit/stats.
- services/  Third-party service UI icons, such as Discord Nitro.
- spheres/   Sphere icons.

The browser tab icon is loaded from:
assets/icons/app/favicon.png

A cache-busting query is used in index.html, so replacing the PNG should refresh more reliably.
If your browser still shows the old icon, close the tab or clear the favicon cache.
